![image](https://github.com/Git-XIXI/xixi-search/assets/148945647/e46f547b-6282-4d31-a586-3449c6de8c2c)# xixi-search
基于 Spring Boot+Elastic Stack+Vue3 的一站式通用聚合搜索平台。用户可在同一页面集中搜索出如文章、图片视频等不同来源、不同类型的内容，提升搜索体验。
